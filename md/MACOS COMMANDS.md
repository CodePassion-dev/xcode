# macOS Commands

## System

```bash
# Check macOS version
% sw_vers
ProductName:		macOS
ProductVersion:		15.0.1
BuildVersion:		24A348
```

```bash
# Check macOS updates
$ softwareupdate -l
```

```bash
# Name your Mac
scutil --set ComputerName "the-name"
scutil --set LocalHostName "the-name"
```
